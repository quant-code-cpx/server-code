## 2. 核心业务规则

| 规则编号 | 规则 | 来源 | 测试影响 |
| -------- | ---- | ---- | -------- |
| R-EX01 | runId 必填，字符串 | DTO @IsString @IsNotEmpty | 缺失/空应 400 |
| R-EX02 | factorId 必填，字符串 | DTO @IsString @IsNotEmpty | 缺失/空应 400 |
| R-EX03 | portfolioId 必填，字符串 | DTO @IsString @IsNotEmpty | 缺失/空应 400 |
| R-EX04 | startDate/endDate 格式 YYYYMMDD | DTO @Matches(/^\d{8}$/) | 格式错误应 400 |
| R-EX05 | tradeDate 格式 YYYYMMDD | DTO @Matches(/^\d{8}$/) | 格式错误应 400 |
| R-EX06 | stock-list columns 必须在白名单内 | DTO @IsIn(validColumns) | 无效应 400 |
| R-EX07 | stock-list columns 最大 25 列 | DTO @ArrayMaxSize(25) | 超限应 400 |
| R-EX08 | factor-screening conditions 必填 | 继承 FactorScreeningDto @IsArray | 缺失应 400 |
| R-EX09 | factor-screening tradeDate 必填 | 继承 FactorScreeningDto @Matches | 缺失/格式错误应 400 |
| R-EX10 | 回测/持仓导出需验证资源归属 | service 层 userId 校验 | 他人资源应 403 |
| R-EX11 | 因子需存在且已启用 | service 层因子状态检查 | 不存在/禁用应 404 |
| R-EX12 | 所有端点需 JWT 认证 | @UseGuards(JwtAuthGuard) | 无 Token 应 401 |
