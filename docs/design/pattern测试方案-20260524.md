## 2. 核心业务规则

| 规则编号 | 规则 | 来源 | 测试影响 |
| -------- | ---- | ---- | -------- |
| R-PT01 | tsCode 必填，字符串 | DTO @IsString | 缺失应 400 |
| R-PT02 | startDate 必须是 8 位数字（YYYYMMDD） | DTO @Matches /^\d{8}$/ | 格式错误应 400 |
| R-PT03 | endDate 必须是 8 位数字（YYYYMMDD） | DTO @Matches /^\d{8}$/ | 格式错误应 400 |
| R-PT04 | algorithm 可选，枚举 NED/DTW，默认 NED | DTO @IsEnum(PatternAlgorithm) | 无效应 400 |
| R-PT05 | topK 可选，1~100，默认 20 | DTO @IsInt @Min(1) @Max(100) | 超范围应 400 |
| R-PT06 | scope 可选，枚举 ALL/INDEX，默认 ALL | DTO @IsEnum(PatternScope) | 无效应 400 |
| R-PT07 | lookbackYears 可选，1~20，默认 5 | DTO @IsInt @Min(1) @Max(20) | 超范围应 400 |
| R-PT08 | excludeSelf 可选，布尔，默认 true | DTO @IsBoolean | 非布尔应 400 |
| R-PT09 | series 必须是数组，至少 5 个数字 | DTO @IsArray @ArrayMinSize(5) @IsNumber | 不足 5 个应 400 |
| R-PT10 | 无 JWT 守卫，所有端点公开访问 | Controller 无 @UseGuards | 无 Token 仍可访问 |
