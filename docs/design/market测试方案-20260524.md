## 2. 核心业务规则

| 规则编号 | 规则 | 来源 | 测试影响 |
| -------- | ---- | ---- | -------- |
| R-MK01 | trade_date 格式 YYYYMMDD | DTO @Matches(/^\d{8}$/) | 格式错误应 400 |
| R-MK02 | content_type 必须是 INDUSTRY/CONCEPT/REGION | DTO @IsEnum | 无效应 400 |
| R-MK03 | days 范围因端点而异（hsgt-flow: 1~365, vol-overview/sentiment-trend/money-flow-trend: 5~120, sector-flow-trend/stock-flow-detail: 5~60） | DTO @Min @Max | 超范围应 400 |
| R-MK04 | limit 范围因端点而异（sector-flow: 1~100, sector-ranking: 1~500, sector-flow-ranking/main-flow-ranking: 1~100, top-movers: 1~100, concept-list pageSize: 1~100, concept-members pageSize: 1~200, sector-top-bottom top_n: 1~20） | DTO @Min @Max | 超范围应 400 |
| R-MK05 | ts_code 必填（sector-flow-trend, stock-flow-detail） | DTO @IsString @IsNotEmpty | 缺失/空应 400 |
| R-MK06 | tsCode 必填（concept-members） | DTO @IsString @IsNotEmpty | 缺失/空应 400 |
| R-MK07 | dim 必填且为枚举值（top-movers: gain/loss/amplitude/amount） | DTO @IsEnum | 缺失/无效应 400 |
| R-MK08 | period 必须是枚举值（index-trend: 1m/3m/6m/1y/3y, valuation-trend: 3m/6m/1y/3y/5y, hsgt-trend: 1m/3m/6m/1y） | DTO @IsEnum | 无效应 400 |
| R-MK09 | sort_by 必须是枚举值 | DTO @IsEnum | 无效应 400 |
| R-MK10 | order 必须是 asc/desc | DTO @IsEnum | 无效应 400 |
| R-MK11 | ts_codes 必须是字符串数组 | DTO @IsArray @IsString({ each: true }) | 非数组应 400 |
| R-MK12 | sparkline_period 必须是 1m/3m/6m/1y/3y | DTO @IsEnum | 无效应 400 |
