## 2. 核心业务规则

| 规则编号 | 规则 | 来源 | 测试影响 |
| -------- | ---- | ---- | -------- |
| R-F01 | factorName 必填 | DTO @IsString @IsNotEmpty | 缺失应 400 |
| R-F02 | 日期格式 YYYYMMDD | DTO @Matches | 格式错误应 400 |
| R-F03 | factorNames 相关性最少 2 个 | DTO @ArrayMinSize(2) | 少于应 400 |
| R-F04 | 自定义因子 name 2~50 字符，小写字母开头 | DTO @Length(2,50) @Matches | 不符合应 400 |
| R-F05 | category 必须是 FactorCategory 枚举 | DTO @IsEnum | 无效应 400 |
| R-F06 | 所有端点需 JWT | @UseGuards(JwtAuthGuard) | 无 Token 应 401 |
| R-F07 | admin 端点需 ADMIN 角色 | @Roles(UserRole.ADMIN) | 普通用户应 403 |
| R-F08 | forwardDays 1~60 | DTO @Min(1) @Max(60) | 超范围应 400 |
| R-F09 | quantiles 3~10 | DTO @Min(3) @Max(10) | 超范围应 400 |
| R-F10 | bins 10~100 | DTO @Min(10) @Max(100) | 超范围应 400 |
| R-F11 | topN 5~100 | DTO @Min(5) @Max(100) | 超范围应 400 |
| R-F12 | initialCapital >= 10000 | DTO @Min(10000) | 少于应 400 |
| R-F13 | expression 最长 500 字符 | DTO @MaxLength(500) | 超长应 400 |
