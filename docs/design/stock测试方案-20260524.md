## 2. 核心业务规则

| 规则编号 | 规则 | 来源 | 测试影响 |
| -------- | ---- | ---- | -------- |
| R-ST01 | code 必填，最大 16 字符 | DTO @IsString @IsNotEmpty @MaxLength(16) | 缺失/空应 400 |
| R-ST02 | keyword 必填，最大 50 字符 | DTO @IsString @IsNotEmpty @MaxLength(50) | 缺失/空应 400 |
| R-ST03 | exchange 必须是 SSE/SZSE/BSE | DTO @IsIn | 无效应 400 |
| R-ST04 | pageSize 1~100 | DTO @Min(1) @Max(100) | 超范围应 400 |
| R-ST05 | days 范围因端点而异 | DTO @Min @Max | 超范围应 400 |
| R-ST06 | 选股策略端点需 JWT | @CurrentUser() | 无 Token 应 401 |
| R-ST07 | 创建策略 name 必填 1~50 字符 | DTO @MinLength(1) @MaxLength(50) | 空/超长应 400 |
| R-ST08 | filters 必须是对象 | DTO @IsObject @ValidateNested | 非对象应 400 |
| R-ST09 | sortBy 必须是枚举值 | DTO @IsEnum | 无效应 400 |
| R-ST10 | 日期格式 YYYYMMDD | DTO @Matches | 格式错误应 400 |
