## 2. 核心业务规则

| 规则编号 | 规则 | 来源 | 测试影响 |
| -------- | ---- | ---- | -------- |
| R-B01 | strategyType 必须是枚举值 | DTO @IsEnum | 无效类型应 400 |
| R-B02 | startDate/endDate 格式 YYYYMMDD | DTO @Matches | 格式错误应 400 |
| R-B03 | initialCapital >= 1000 | DTO @Min(1000) | 少于应 400 |
| R-B04 | strategyConfig 必须是对象 | DTO @IsObject | 非对象应 400 |
| R-B05 | 所有端点需 JWT | @UseGuards(JwtAuthGuard) | 无 Token 应 401 |
| R-B06 | comparison strategies 2~10 个 | DTO @ArrayMinSize(2) @ArrayMaxSize(10) | 超范围应 400 |
| R-B07 | walk-forward inSampleDays 60~2520 | DTO @Min(60) @Max(2520) | 超范围应 400 |
| R-B08 | walk-forward outOfSampleDays 20~504 | DTO @Min(20) @Max(504) | 超范围应 400 |
| R-B09 | monte-carlo numSimulations 100~10000 | DTO @Min(100) @Max(10000) | 超范围应 400 |
| R-B10 | pageSize 1~100 | DTO @Min(1) @Max(100) | 超范围应 400 |
