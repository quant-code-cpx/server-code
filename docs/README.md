## 文档编写约定

1. **设计文档**（`design/`）面向 AI 代码生成模型，需包含：精确的文件路径、TypeScript 接口签名、Prisma Schema、SQL 逻辑和算法伪代码
2. **文件命名**：中文命名，格式 `<模块名>-后端设计.md`（功能设计）、`<模块名>-需求.md`（需求稿）
3. **状态标记**：✅ 已实现 / 🔧 待实现 / 📋 需求稿 / 🗓️ 规划中
4. **新增文档**时请同步更新本 README
