## 2. 核心业务规则

| 规则编号 | 规则 | 来源 | 测试影响 |
| -------- | ---- | ---- | -------- |
| R-HM01 | trade_date 格式为 YYYYMMDD | DTO @Matches(/^\d{8}$/) | 格式错误应 400 |
| R-HM02 | group_by 必须是 industry/index/concept | DTO @IsIn | 无效应 400 |
| R-HM03 | industry_source 必须是 stock_basic/sw_l1 | DTO @IsIn | 无效应 400 |
| R-HM04 | limit 范围 1~5000 | DTO @Min(1) @Max(5000) | 超范围应 400 |
| R-HM05 | snapshot/trigger 需 SUPER_ADMIN 角色 | @UseGuards(RolesGuard) @Roles(SUPER_ADMIN) | 非管理员应 403 |
| R-HM06 | snapshot/history trade_date 必填 | DTO @IsNotEmpty | 缺失应 400 |
| R-HM07 | snapshot/history group_by 必须是枚举值 | DTO @IsIn | 无效应 400 |
