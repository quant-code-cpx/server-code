## 1. 本轮目标

- 验证 PostgreSQL 可独立保存并恢复 Run 权威状态、Step、checkpoint 与完整事件序列。

| 项目 | 结论 |

| 业务场景 | 用户请求形成可恢复 Run；Worker 以 lease 推进；页面按 sequence 重放事件 |
