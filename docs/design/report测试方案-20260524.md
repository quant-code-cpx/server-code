## 2. 核心业务规则

| 规则编号 | 规则 | 来源 | 测试影响 |
| -------- | ---- | ---- | -------- |
| R-RP01 | 所有端点需 JWT 认证 | @UseGuards(JwtAuthGuard) | 无 Token 应 401 |
| R-RP02 | backtest 端点 runId 必填 | @IsString | 缺失应 400 |
| R-RP03 | stock 端点 tsCode 必填 | @IsString | 缺失应 400 |
| R-RP04 | portfolio 端点 portfolioId 必填 | @IsString | 缺失应 400 |
| R-RP05 | strategy-research 端点 backtestRunId 必填 | @IsString | 缺失应 400 |
| R-RP06 | format 必须是 JSON/HTML/PDF | @IsEnum(ReportFormatEnum) | 非法值应 400 |
| R-RP07 | page >= 1 | @Min(1) | 小于 1 应 400 |
| R-RP08 | pageSize 1~100 | @Min(1) @Max(100) | 超范围应 400 |
| R-RP09 | sections 必须是对象 | @IsObject @ValidateNested | 非对象应 400 |
| R-RP10 | reportId 必填 | @Body body: { reportId: string } | 缺失应 400 或 service 报错 |
