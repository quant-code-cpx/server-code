## 3. 业务理解

### 3.1 业务场景

因子市场是量化研究核心模块，服务于：

1. **因子发现**：用户浏览因子库，按分类/IC/覆盖率筛选高质量因子
2. **因子分析**：对单个因子做 IC 时间序列、分层回测、衰减分析、截面分布统计
3. **多因子研究**：因子相关性矩阵、正交化、Fama-MacBeth 回归检验
4. **因子选股**：多条件组合筛选股票
5. **自定义因子**：表达式引擎创建/试算/管理自定义因子
6. **因子回测**：因子筛选条件一键回测 + 归因分析
7. **组合优化**：MVO/最小方差/风险平价/最大分散化生成最优权重
8. **管理运维**：预计算触发、回补、状态查询

## 6. 风险分析

| 风险 | 业务影响 | 测试策略 | 优先级 |
| ---- | -------- | -------- | ------ |
| IC/IR 计算错误 | 错误评估因子有效性，导致选入无效因子 | 手算小样本 IC 比对 | P0 |
| 分层回测收益计算偏差 | 多空收益被高估/低估 | 手算等权组合收益比对 | P0 |
| 因子筛选 AND 逻辑错误 | 选出不应入选的股票 | 构造条件边界交叉验证 | P0 |
| 自定义因子表达式注入 | SQL 注入或非法计算导致系统异常 | 注入变体测试 | P0 |
| 正交化/Fama-MacBeth 矩阵运算错误 | 高级量化分析结论偏差 | 与 Python/R 独立计算比对 | P1 |
| 组合优化权重不满足约束 | 生成的权重无效 | 验证 sum(w)=1 且 w∈[minW,maxW] | P1 |
| 分页 total 与实际 items 不一致 | 前端翻页显示异常 | 多页交叉验证 | P1 |
| 权限控制缺失 | 普通用户访问管理接口 | 无 token / 普通 token 测试 | P1 |
| 预计算/回填任务重复触发 | 队列资源浪费 | 并发触发验证幂等 | P2 |
